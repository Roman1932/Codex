# aso

Use when auditing or optimizing an App Store or Google Play listing.

## Main jobs

- audit app title, subtitle, description, screenshots, icon, reviews, and conversion elements
- improve keyword targeting for app marketplaces
- rewrite listing copy for clarity and conversion
- identify visual and positioning weaknesses
- plan listing tests

## Inputs

- app name and store URL
- target country and language
- target keywords
- competitor app links
- current conversion/download data
- app category and monetization model

## Output

- ASO diagnosis
- priority fixes
- keyword recommendations
- copy recommendations
- screenshot/visual recommendations
- testing ideas

## Related skills

product-marketing, copywriting, customer-research, competitors, analytics
