# offers

Use when designing, constructing, or improving the offer: what is being sold, how value is framed, what is included, and why the buyer should act.

## Main jobs

- improve offer clarity
- strengthen value framing
- define bonuses and bundles
- reduce perceived risk
- handle objections
- improve price-value perception
- package the offer for conversion

## Inputs

- product/service
- target buyer
- current offer
- price
- bonuses/guarantees
- objections
- proof points

## Output

- improved offer structure
- value proposition
- bonuses/bundles
- objection handling
- risk reversal ideas
- CTA and framing notes

product-marketing, pricing, copywriting, cro, customer-research
