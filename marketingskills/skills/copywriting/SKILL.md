# copywriting

Use when writing, rewriting, or improving marketing copy for pages, ads, emails, product descriptions, landing pages, and offers.

## Main jobs

- write page copy
- create headlines and CTAs
- structure landing pages
- adapt copy to audience awareness
- sharpen benefits and proof
- handle objections

## Inputs

- product context
- target audience
- page/channel type
- offer
- tone
- proof points
- desired action

## Output

- structured copy
- headline options
- CTA options
- objection-handling sections
- benefit-focused messaging
- revision notes
