# onboarding

Use when optimizing post-signup onboarding, user activation, first-run experience, and time-to-value.

## Main jobs

- map activation journey
- identify onboarding friction
- define aha moment
- improve first-run steps
- plan onboarding emails/tooltips/checklists
- measure activation and drop-off

## Inputs

- product and user type
- signup flow
- current onboarding steps
- activation metric
- drop-off points
- user feedback

## Output

- onboarding diagnosis
- improved flow
- activation checklist
- messaging recommendations
- experiment ideas
- metrics

## Related skills

signup, emails, analytics, churn-prevention, product-marketing
