# public-relations

Use when working on PR, earned media, press coverage, journalist outreach, or media strategy.

## Main jobs

- define PR angle
- identify media targets
- write pitch emails
- create press release outline
- plan founder/story angles
- prepare media kit checklist
- track outreach

## Inputs

- company/product
- news hook
- target audience
- proof points
- geography/industry
- media targets if known

## Output

- PR angle
- pitch
- press release structure
- media list criteria
- outreach sequence
- tracking plan

## Related skills

product-marketing, launch, co-marketing, cold-email, social
