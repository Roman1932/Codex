# ai-seo

Use when optimizing content for AI search engines, AI citations, LLM visibility, AI Overviews, Perplexity, ChatGPT search, Claude, Gemini, Copilot, llms.txt, OKF, and agent-readable websites.

## Main jobs

- audit AI visibility
- test key queries across AI platforms
- improve extractability of content
- strengthen citation-worthiness
- plan AI-readable content blocks
- evaluate robots.txt AI crawler access
- add structured machine-readable context where useful

## Inputs

- target queries
- target platforms
- important pages
- competitors cited by AI
- schema status
- robots.txt status
- third-party mentions

## Output

- AI visibility audit
- content restructuring plan
- citation and authority plan
- llms.txt / pricing.md / OKF recommendations
- monitoring plan

seo-audit, schema, content-strategy, competitors, copywriting
