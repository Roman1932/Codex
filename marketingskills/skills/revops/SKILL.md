# revops

Use when working on revenue operations, lead lifecycle management, CRM process, marketing-to-sales handoff, and pipeline tracking.

## Main jobs

- define lifecycle stages
- map lead sources
- improve CRM fields and routing
- align marketing and sales handoff
- define pipeline metrics
- standardize reporting

## Inputs

- CRM/tool stack
- lead sources
- funnel stages
- sales process
- current reporting
- business goal

## Output

- lifecycle map
- CRM field recommendations
- handoff rules
- pipeline metrics
- reporting structure
- implementation checklist

## Related skills

analytics, sales-enablement, prospecting, cold-email, product-marketing
