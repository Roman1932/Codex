# context-foundation

Use first in any new marketing project to create the reusable product and market context used by all other skills.

## Main jobs

- define product overview
- define ideal customer profile
- map audience and personas
- capture problems and pain points
- describe competitive landscape
- define differentiation
- document objections
- capture customer language
- record proof points and goals

## Inputs

- product or store
- audience
- market and language
- competitors
- offer
- customer feedback
- current metrics

## Output

- product overview
- ICP
- personas
- pain points
- differentiation
- objections
- customer language
- proof points
- goals

## Related skills

All other marketing skills should reference this context before execution.
