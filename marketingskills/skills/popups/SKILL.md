# popups

Use when creating or optimizing popups, modals, overlays, slide-ins, and banners for conversion.

## Main jobs

- choose popup goal
- define trigger and timing
- write popup copy
- reduce annoyance
- improve offer and CTA
- plan targeting and suppression rules
- measure impact

## Inputs

- page/site context
- popup goal
- target audience
- offer
- traffic source
- current conversion rate
- existing popup rules

## Output

- popup strategy
- copy variants
- trigger rules
- targeting/suppression rules
- design notes
- test plan

## Related skills

cro, offers, copywriting, emails, lead-magnets
