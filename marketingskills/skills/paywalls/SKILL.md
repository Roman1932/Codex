# paywalls

Use when creating or optimizing in-app paywalls, upgrade screens, upsell modals, and feature gates.

## Main jobs

- clarify upgrade value
- improve plan comparison
- reduce upgrade friction
- test paywall copy and layout
- handle objections
- improve conversion from free/trial to paid

## Inputs

- product and pricing
- current paywall/upgrade screen
- user segment
- conversion rate
- objections
- plan features

## Output

- paywall diagnosis
- copy recommendations
- layout recommendations
- test ideas
- metrics and tracking plan

## Related skills

pricing, offers, cro, onboarding, analytics
