# signup

Use when optimizing signup, registration, account creation, or trial activation flows.

## Main jobs

- diagnose signup friction
- simplify forms
- improve CTA and value framing
- reduce trust/anxiety blockers
- define activation tracking
- create test ideas

## Inputs

- signup flow URL/screens
- current conversion rate
- traffic source
- required fields
- post-signup flow
- user objections

## Output

- signup diagnosis
- friction fixes
- form recommendations
- copy improvements
- tracking plan
- A/B test ideas

## Related skills

cro, onboarding, analytics, copywriting, ab-testing
