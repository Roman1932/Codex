# emails

Use when creating or optimizing email sequences, drip campaigns, automated flows, and lifecycle emails.

## Main jobs

- build welcome flows
- create abandoned cart flows
- create post-purchase flows
- write onboarding emails
- create winback and reactivation emails
- improve subject lines and CTAs
- map lifecycle stages

## Inputs

- business model
- audience
- email goal
- trigger event
- current sequence if any
- offer and proof points
- tone/language

## Output

- email flow map
- subject lines
- email sequence
- CTA logic
- segmentation notes
- measurement plan

## Related skills

product-marketing, copywriting, offers, churn-prevention, analytics
