# video

Use when creating, generating, or producing video content using AI tools or programmatic frameworks.

## Main jobs

- create video concepts
- write hooks and scripts
- generate video prompts
- plan shot structure
- adapt for TikTok/Reels/Shorts/ads
- create storyboard and production notes

## Inputs

- product/brand context
- platform and format
- target audience
- video goal
- available assets
- tone/style
- duration

## Output

- video concept
- hook options
- script
- shot list
- AI video prompt
- platform notes
- CTA

social, image, ad-creative, copywriting, product-marketing
