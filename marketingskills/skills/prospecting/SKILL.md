# prospecting

Use when finding, qualifying, and building a list of prospects for outreach.

## Main jobs

- define prospect criteria
- build prospect list structure
- qualify accounts/people
- identify outreach angles
- prepare enrichment fields
- prioritize prospects

## Inputs

- target customer profile
- industry/market
- geography
- offer
- outreach goal
- available data sources

## Output

- prospect criteria
- list fields
- qualification rules
- outreach angles
- prioritization framework
- next-step workflow

## Related skills

product-marketing, cold-email, co-marketing, revops, sales-enablement
