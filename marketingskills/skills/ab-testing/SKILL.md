# ab-testing

Use when planning, designing, implementing, or evaluating A/B tests, split tests, experiments, multivariate tests, and growth experimentation programs.

## Main jobs

- define hypothesis
- choose primary, secondary, and guardrail metrics
- estimate sample size and test duration
- design variants
- build experiment backlog
- prioritize with ICE scoring
- interpret results without false positives

## Inputs

- current conversion rate
- traffic volume
- page or flow to test
- proposed change
- business goal
- analytics/testing tool

## Output

- hypothesis
- test design
- variant plan
- metrics
- sample-size guidance
- QA checklist
- result interpretation framework

## Related skills

analytics, cro, copywriting
