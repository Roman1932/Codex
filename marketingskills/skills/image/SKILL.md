# image

Use when creating, generating, editing, or optimizing images for marketing: blog heroes, social graphics, product visuals, thumbnails, ad creatives, and promotional assets.

## Main jobs

- define image concept
- write image generation prompts
- adapt visuals to platform specs
- create visual variants
- improve clarity and conversion value
- align image with offer and audience

## Inputs

- product/brand context
- platform and format
- image purpose
- target audience
- visual style
- required text or no-text rule
- source images if available

## Output

- image brief
- prompt
- layout direction
- platform-specific notes
- variant ideas
- quality checklist

## Related skills

ad-creative, social, video, copywriting, product-marketing
