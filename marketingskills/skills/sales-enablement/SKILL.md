# sales-enablement

Use when creating sales collateral, pitch decks, one-pagers, objection handling docs, demo scripts, and battlecards.

## Main jobs

- create sales one-pagers
- build objection handling
- prepare demo scripts
- create competitor battlecards
- align sales messaging with positioning
- support outreach and closing

## Inputs

- product context
- buyer persona
- sales situation
- common objections
- competitors
- proof points

## Output

- sales collateral
- objection responses
- demo script
- battlecard
- pitch structure
- follow-up notes

## Related skills

product-marketing, competitors, competitor-profiling, revops, copywriting
