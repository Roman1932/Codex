# cro

Use when optimizing, improving, or increasing conversions on marketing pages, landing pages, homepages, pricing pages, feature pages, product pages, forms, or contact flows.

## Main jobs

- diagnose conversion problems
- improve value proposition clarity
- improve headlines and CTA hierarchy
- identify friction points
- improve trust signals and social proof
- create test ideas

## Inputs

- page URL or copy
- page type
- conversion goal
- traffic source
- current conversion rate
- known user objections

## Output

- quick wins
- high-impact changes
- test ideas
- copy alternatives
- friction analysis
- priority action plan

## Related skills

signup, popups, copywriting, ab-testing, analytics
