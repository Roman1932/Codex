# site-architecture

Use when planning, mapping, or restructuring website hierarchy, navigation, URL structure, and internal links.

## Main jobs

- map current site structure
- plan category/page hierarchy
- improve navigation
- define URL patterns
- plan internal linking
- reduce orphan and duplicate pages
- support SEO and UX

## Inputs

- site URL or sitemap
- business/site type
- target topics/categories
- current navigation
- SEO goals
- target markets/languages

## Output

- proposed architecture
- URL structure
- navigation recommendations
- internal linking plan
- page types
- implementation priorities
