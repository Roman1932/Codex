# schema

Use when adding, fixing, or optimizing schema markup and structured data.

## Main jobs

- choose correct schema types
- audit existing structured data
- create JSON-LD
- validate rich result eligibility
- improve entity clarity
- support SEO and AI extractability

## Inputs

- page type
- URL or page content
- business/product data
- current schema if any
- target rich result
- CMS/platform

## Output

- schema recommendation
- JSON-LD draft
- validation checklist
- implementation notes
- common error warnings

## Related skills

seo-audit, ai-seo, product-marketing, site-architecture
