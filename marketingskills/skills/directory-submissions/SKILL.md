# directory-submissions

Use when submitting a product to startup, SaaS, AI, agent, MCP, no-code, marketplace, or review directories.

## Main jobs

- choose relevant directories
- prepare submission copy
- adapt descriptions to directory requirements
- define categories and tags
- track submissions and follow-ups
- use directories for SEO, discovery, and lead generation

## Inputs

- product name and URL
- one-line description
- longer description
- target category
- logo/screenshots if needed
- target market

## Output

- directory shortlist
- submission descriptions
- tags/categories
- tracking sheet structure
- priority order
- follow-up guidance

product-marketing, copywriting, seo-audit, launch, public-relations
