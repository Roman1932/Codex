# churn-prevention

Use when reducing churn, improving cancellation flows, creating save offers, recovering failed payments, or improving retention messaging.

## Main jobs

- diagnose churn causes
- segment churn risk
- design cancellation surveys
- build save offers
- create winback flows
- recover failed payments
- design retention experiments

## Inputs

- churn rate and period
- cancellation reasons
- pricing/plans
- customer segments
- usage or purchase frequency
- support/refund reasons
- existing retention flows

## Output

- churn diagnosis
- high-risk segments
- cancellation-flow recommendations
- save offers
- recovery sequence
- winback campaign
- measurement plan

## Related skills

product-marketing, emails, analytics, offers, customer-research, onboarding
