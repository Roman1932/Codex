# copy-editing

Use when editing, reviewing, refreshing, or improving existing marketing copy.

## Main jobs

- improve clarity
- remove fluff and weak claims
- strengthen conversion logic
- fix tone and structure
- sharpen value propositions
- make copy more specific and credible

## Inputs

- current copy
- page or channel type
- target audience
- product context
- desired tone
- conversion goal

## Output

- edited copy
- explanation of key changes
- stronger headlines/CTAs
- clarity and credibility fixes
- optional before/after notes

## Related skills

copywriting, product-marketing, cro, marketing-psychology
