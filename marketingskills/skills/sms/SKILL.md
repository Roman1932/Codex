# sms

Use when planning, building, or optimizing SMS/MMS marketing flows and campaigns.

## Main jobs

- create SMS welcome flows
- create abandoned cart messages
- write promotional SMS
- plan compliance-conscious messaging
- define segmentation and frequency
- measure SMS performance

## Inputs

- audience
- country/compliance context
- offer
- trigger event
- tone
- current flow if any
- target metric

## Output

- SMS flow
- message copy
- segmentation rules
- timing/frequency
- opt-out/compliance notes
- measurement plan

## Related skills

emails, offers, copywriting, churn-prevention, analytics
