# behavioral-marketing

Use when applying behavioral science and decision-making principles to marketing, conversion, messaging, pricing, and offers.

## Main jobs

- improve persuasion logic
- reduce buyer uncertainty
- strengthen perceived value
- improve trust and motivation
- diagnose decision blockers
- create ethical test ideas

## Inputs

- page, offer, or copy
- audience
- decision context
- conversion goal
- objections
- proof elements

## Output

- decision-friction diagnosis
- principle-based recommendations
- copy/UX improvements
- test ideas
- risk notes

## Related skills

copywriting, cro, offers, pricing, ab-testing
