# ad-creative

Use when generating, iterating, or scaling ad creative for paid channels: headlines, descriptions, primary text, creative angles, hooks, and ad variations.

## Main jobs

- define ad angles
- generate platform-specific ad copy
- validate character limits
- create creative testing batches
- analyze winning and losing patterns from performance data
- produce CSV-ready ad variations

## Inputs

- platform and format
- product/offer
- audience and intent stage
- existing ad performance data
- compliance or brand constraints

## Output

- angles
- headlines
- descriptions
- primary text
- iteration report
- upload-ready structure

## Related skills

ads, copywriting, ab-testing, marketing-psychology, copy-editing
