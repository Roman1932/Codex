# programmatic-seo

Use when creating SEO-driven pages at scale using templates and data.

## Main jobs

- identify scalable SEO page patterns
- define page template
- map data fields
- prevent thin/duplicate content
- plan internal linking
- define indexation rules
- create quality controls

## Inputs

- keyword pattern
- data source
- page type
- site structure
- target market/language
- examples of pages
- SEO goal

## Output

- page pattern
- template structure
- data schema
- quality rules
- internal linking plan
- launch checklist

## Related skills

seo-audit, site-architecture, schema, content-strategy, ai-seo
