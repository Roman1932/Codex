# marketing-plan

Use when creating a comprehensive marketing plan for a client, company, product, or project.

## Main jobs

- define growth objective
- analyze audience and positioning
- choose channels
- plan campaigns
- prioritize initiatives
- set KPIs and budget
- create timeline

## Inputs

- product/company context
- target audience
- current traction
- goals
- budget/resources
- market and competitors
- timeframe

## Output

- marketing strategy
- channel plan
- campaign roadmap
- budget allocation
- KPIs
- timeline
- next actions
