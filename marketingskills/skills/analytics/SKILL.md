# analytics

Use when setting up, improving, or auditing analytics tracking and measurement: GA4, GTM, conversion tracking, event tracking, UTM strategy, attribution, Mixpanel, Segment, or data quality.

## Main jobs

- create tracking plan
- define events and properties
- standardize naming conventions
- implement GA4/GTM logic
- validate event firing
- define conversions
- prevent duplicate or dirty data
- handle privacy and consent requirements

## Inputs

- tools used
- key actions to track
- business decisions supported by data
- implementation owner
- consent/privacy requirements
- current tracking setup

## Output

- tracking plan
- event taxonomy
- conversion map
- GTM/GA4 checklist
- UTM naming rules
- validation checklist

ab-testing, seo-audit, cro, revops
