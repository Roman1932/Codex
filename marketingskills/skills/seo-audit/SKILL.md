# seo-audit

Use when auditing, reviewing, or diagnosing SEO issues on a site.

## Main jobs

- audit crawlability and indexation
- check technical SEO foundations
- review titles, metas, headings, content, internal links
- identify content quality issues
- check international SEO if relevant
- prioritize fixes

## Inputs

- site URL
- site type
- target market/language
- priority keywords/topics
- current traffic/ranking issue
- Search Console/analytics access if available

## Output

- executive summary
- technical findings
- on-page findings
- content findings
- prioritized action plan
- quick wins

schema, ai-seo, content-strategy, site-architecture, analytics
