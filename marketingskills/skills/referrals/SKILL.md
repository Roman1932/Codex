# referrals

Use when creating, optimizing, or analyzing referral, affiliate, or word-of-mouth programs.

## Main jobs

- design referral program
- define incentive structure
- plan invite/referral flow
- create referral messaging
- evaluate affiliate fit
- measure referral performance

## Inputs

- product and audience
- current customer base
- incentive options
- margin/CPA constraints
- existing referral flow
- target metric

## Output

- referral strategy
- incentive model
- messaging
- tracking plan
- launch checklist
- optimization ideas

## Related skills

product-marketing, offers, emails, community-marketing, analytics
