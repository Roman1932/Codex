# MarketingSkills working copy

Source repository: `coreyhaines31/marketingskills`

This folder stores a local working copy of the MarketingSkills skill list for future AI/agent marketing work.

## Structure

- `skills/` — one markdown file per skill
- `SKILLS_INDEX.md` — full index of skill names and use cases

## Important note

This is a working copy created for practical use inside this repository. The original public source remains:
https://github.com/coreyhaines31/marketingskills

## Recommended workflow

1. Start with `product-marketing` to define the product, ICP, positioning, competitors, customer language, proof points, and goals.
2. Choose a specialist skill depending on the task.
3. Use the skill file as the operating instruction for the AI agent.
4. Save project-specific context and outputs separately.
